# ov-workshop 🐙✨

Octo + Veran 的小工具工作坊。

## 工具列表

| 工具 | 說明 | 使用方式 |
|------|------|----------|
| **Chat Visualizer** | Claude / ChatGPT 對話記錄可視化工具 | 瀏覽器直接打開 `chat-visualizer.html` |
| **JSX Previewer** | 貼入 JSX 代碼即時預覽 | 瀏覽器直接打開 `jsx-previewer.html` |

## Chat Visualizer 功能

- 支援 Claude / ChatGPT 匯出的 JSON
- 1:1 還原對話流程：思考鏈、工具調用、工具結果、Artifact 卡片
- 對話標籤分類 + 消息收藏
- 統計面板：頻率圖表、Artifact 總覽、高頻詞雲
- 匯出 HTML / TXT / JSON
- 自訂顯示名稱

## JSX Previewer 功能

- 貼入 React JSX 代碼即時渲染
- 自動轉換 ES Module 語法（`import/export`），從任何地方複製的 JSX 直接貼上就能跑
- 內建 React 18 + Babel + Tailwind + Google Fonts
- 範例代碼一鍵載入
- Ctrl+Enter 快速執行
- ⛶ 全屏預覽：一鍵在新分頁打開，完整網頁體驗
